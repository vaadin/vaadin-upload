vaadin-upload Code Examples

Overview

The vaadin-upload is a Polymer element for uploading multiple files to server.

Basic Usage

Events

Added files:

    // code var upload = upload || document.querySelector('vaadin-upload'); upload.addEventListener('file-add', function(event) { var file = event.detail.file; var li = document.createElement('li'); li.textContent = file.name + ': ' + file.size + ' bytes'; document.querySelector('#added-files').appendChild(li); }); // end-code
